APP NAME: PARADIGM (Web Version)
Project Proposal: Mindful Habit & Goal Setter (Website)
Core Concept:
A personal and private website designed to help users understand the connection between their
daily habits, sleep patterns, and emotional state. It will function as a digital journal where users
can track their well-being and set meaningful goals based on personal insights, accessible from
any web browser.
Key Problems Addressed:
??? A lack of awareness of how specific habits impact mood, energy, and stress.
??? Difficulty in consistently tracking personal patterns (e.g., sleep quality, emotional
triggers).
??? The struggle to set and stick to relevant, personalized goals that genuinely improve
well-being.
Solution Components (How it Will Work on the Website)
1. Integrated Tracking System:
??? Customizable Habit Tracker: This will remain the same. Users can log the habits they
want to build or change (e.g., exercise, reading, hydration).
??? Wellness Journal: This could be even better on the web! Typing with a keyboard is
easier, so users might write more detailed entries.
??? Sleep Patterns: Hours slept, quality of sleep.
??? Emotional Strain: A simple scale to rate daily stress or mood.
??? Pattern Analysis: The correlation engine will work similarly on the backend, analyzing
data to identify patterns.
2. Insight-Driven Goal Setting:
??? Personalized Goal Suggestions: This core feature will work the same way on the web.
??? "It looks like your emotional strain is lower on days you exercise. Try setting a
goal to work out 3 times this week."
??? "You've reported poor sleep on days with over 4 hours of screen time after 8 PM.
Let's set a goal to unplug earlier."
??? Note on Screen Time: A mobile app can track screen time automatically, but a
web app cannot. The user would need to perform manual entry for this, or a
browser extension could be developed in the future.
??? Goal-Focused View: A clean dashboard will display active goals and progress. The
larger screen of a desktop can provide a richer, more detailed view.

3. Progress & Motivation:
??? Personal Dashboards: It will be easier to display detailed graphs and trends on a larger
screen, allowing users to visualize their mood, sleep, and habit streaks more effectively.
??? Achievement Milestones: The same concept of using badges and awards to boost
motivation.
??? Notifications: Instead of mobile push notifications, the web app would use email
reminders or browser notifications (if the user allows them).
Web App: Pros and Cons
??? Pros:
??? No Installation: Users don't need to install anything; they can use it directly by
entering the URL in a browser.
??? Platform Independent: It will work on any device with a browser (PC, Mac,
tablet).
??? Better for Deep Work: People often use their laptops for serious planning and
reflection, making it ideal for writing journal entries or analyzing dashboards.
??? Faster Development (MVP): A basic Minimum Viable Product (MVP) for a web
app can often be built faster than for a mobile app.
??? Cons:
??? Less "In-the-moment": A mobile phone is always accessible for a quick "mood
check-in." For the web version, the user must actively open the site.
??? Lack of Automatic Tracking: Features like screen time cannot be tracked
automatically.
??? Notifications: Browser notifications are often blocked by users and are generally
not as effective as mobile push notifications.
Implementation Approach
Phase 1: Core Features (3 months)
??? Basic manual tracking forms for habits, sleep, and emotional state.
??? A manual goal-setting feature.
??? A simple dashboard to view progress.
??? Example Tech Stack:
??? Frontend: React.js or Vue.js
??? Backend: Node.js (Express.js) or Python (Django)
??? Database: PostgreSQL or MongoDB

Next Steps
These steps are excellent and just as crucial for the web version.
1. User Research: Talk to potential users to understand how they would want to use this
on the web.
2. Build a Prototype: Before development, create a clickable prototype using a tool like
Figma or simple HTML/CSS to get user feedback.
3. Consult an Expert: Consulting with a behavioral psychologist or wellness coach is still a
critical step.
