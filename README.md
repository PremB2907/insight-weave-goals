# PARADIGM: Mindful Habit & Goal Setter (Web MVP)

🔗 Live Application: https://insight-weave-goals.lovable.app  
📦 Repository: https://github.com/PremB2907/insight-weave-goals  

PARADIGM is a **progressive web application** designed to uncover the hidden connections between your daily habits, sleep patterns, and emotional state. It transforms basic tracking into intelligent, automated insights.

---

## 📌 About the Project

Most habit trackers leave it up to the user to figure out *why* they feel bad on consecutive Tuesdays. **PARADIGM** automatically correlates your actions with your feelings by providing:
- A sophisticated unified **Wellness Journal** for sleep, mood, and stress logging.
- An **Intelligent Correlation Engine** that computes exactly how your specific habits affect your metrics.
- **Insight-Driven Automated Goals** that adaptively suggest goals based on where your statistics are falling behind. 
- A Gamified **Achievement System** powered by global state to seamlessly reward consistency.

This repository fulfills the PARADIGM Phase 1 & 2 project proposal requirements as a fully functional frontend mock.

---

## ✨ Features Implemented

- **Dynamic Goal Setting:** Define goals manually or accept dynamically generated goals based on your actual 7-day trailing data.
- **Correlation Engine:** Real-time analysis of `Sleep → Energy` or `Stress → Mood` using localized mathematical mapping from your journal records.
- **Global Gamification:** Unlock badges (e.g., *Consistent Logger*, *Habit Builder*) as you achieve milestones automatically.
- **Intelligent Global State:** Leverages robust React Context and `localStorage` to ensure a cohesive MVP without needing immediate backend schema mapping.
- **Premium Glassmorphism UX/UI:** Beautiful, animated components utilizing Shadcn UI & Tailwind CSS.

---

## 🛠️ Upgraded Tech Stack

- **Frontend:** React, TypeScript, Vite
- **UI/Components:** Tailwind CSS, `shadcn/ui`, Radix UI, `lucide-react`
- **State Management:** React Global Context API & browser `localStorage`
- **Alerts & Visuals:** `sonner` Toasts

## 🚀 Running Locally

Clone the repository and run the development server to experience the dynamic gamification logic firsthand:

```bash
git clone https://github.com/PremB2907/insight-weave-goals.git
cd insight-weave-goals
npm install
npm run dev
```
